## 한국어 요지

Claude Code의 사용량 관리에 대한 팁입니다. Claude 생태계는 모든 사용자가 동일한 쿼터 풀을 공유하므로, 웹 인터페이스에서 오랫동안 대화를 나눈 경우 터미널에서 코딩을 시작할 때 이미 사용 가능한 쿼터가 상당히 줄어 있을 수 있습니다. 이를 인지하고 `/usage`와 같은 관련 명령어를 통해 쿼터 현황을 주기적으로 확인하는 것이 중요합니다. 이 명령어들은 쿼터의 남은 양을 파악하는 데 도움을 주며, 사용자는 이를 바탕으로 웹과 터미널에서의 Claude Code 사용량을 효율적으로 조절할 수 있습니다.
