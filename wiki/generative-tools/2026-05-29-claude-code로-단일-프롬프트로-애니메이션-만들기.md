## 한국어 요지

Claude Code를 활용하면 단일 프롬프트만으로도 멋진 애니메이션을 만들 수 있습니다. 먼저 특정 웹사이트를 방문하면, 프리미엄 UI 컴포넌트들을 무료로 이용할 수 있습니다. 이 웹사이트에는 수백 개의 미리 만들어진 애니메이션 컴포넌트가 있어 Claude Code에서 바로 사용할 수 있습니다. 원하는 애니메이션을 선택하고 'Copy Prompt'를 클릭한 뒤 Claude Code를 선택하면 됩니다. 이후 Claude 터미널로 이동하여 프롬프트를 붙여넣기만 하면 애니메이션이 완성됩니다. 디자이너라면 "Doc"이라고 댓글을 남겨 Claude Code 사용법에 대한 상세한 Notion 가이드를 받아볼 수 있습니다. 이 가이드는 DM으로 전송됩니다. 또한, 동료 디자이너들과 이 정보를 공유하는 것도 권장됩니다.
